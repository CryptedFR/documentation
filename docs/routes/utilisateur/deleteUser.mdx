# Supprimer l'utilisateur

Cette route permet de supprimer l'utilisateur authentifié.

## URL

<HttpMethod method="DELETE" route="/v1/user" />

## En-têtes de la requête

| Nom      |        Valeur ou type                  |
|-----------|------------------------------|
| `Content-Type`     | `application/json`            |
| `Authorization`     | `Bearer <token>`            |

## Réponse

### Succès

- **Code HTTP** : `200 OK`

| Nom      |        Valeur ou type                  | Description                                         |
|-----------|------------------------------|-----------------------------------------------------|
| `status`     | `success`            | Status de la réponse.               |
| `statusCode`     | `200`          | Code HTTP de la réponse. |
| `code`     | `S_USER_DELETED`        | Code success.                |

- **Exemple JSON :**

```json
{
  "status": "success",
  "statusCode": 200,
  "code": "S_USER_DELETED"
}
```
