# Accepter une demande d'amis

Cette route permet d'accepter une demande d'amis d'un autre utilisateur.

## URL

<HttpMethod method="PATCH" route="/v1/friendship/accept/:requestId"/>

## En-têtes de la requête

| Nom      |        Valeur ou type                  |
|-----------|------------------------------|
| `Content-Type`     | `application/json`            |
| `Authorization`     | `Bearer <token>`            |

## Corps de la requête

| Champ            | Type     | Requis | Description                           |
|------------------|----------|--------|---------------------------------------|
| `friendId`       | `number` | Oui    | ID de l'utilisateur cible.            |
